# atom-control
Cli tool for atom text editor control

## Install

    git clone https://github.com/pavel-g/atom-control.git
    cd atom-control
    npm install
    npm run build
    node bin/atom-control.js

## Upgrade

    node bin/atom-control.js --upgrade-to=latest
    # or
    node bin/atom-control.js --upgrade-to=1.0.0
